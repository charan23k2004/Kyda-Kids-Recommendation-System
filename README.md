# 🛍️ Kyda Kids Intelligent Recommendation System

> AI-powered recommendation system for **Kyda Kids** that improves production planning by forecasting seasonal demand, analyzing customer sentiment, and generating intelligent product recommendations.

## 📖 Business Problem
Kyda Kids needs accurate demand forecasting and actionable recommendations to reduce overproduction, optimize inventory, and improve customer satisfaction.

## ✨ Features
- Hybrid Recommendation System
- Content-Based Filtering
- Collaborative Filtering
- ARIMA/SARIMA Demand Forecasting
- VADER Sentiment Analysis
- Flask Dashboard
- Interactive Data Visualizations

## 🛠️ Tech Stack
Python • Flask • Pandas • NumPy • Scikit-learn • Statsmodels • NLTK VADER • HTML • CSS • JavaScript • Matplotlib

## 🧠 ML Pipeline
```text
Dataset → Preprocessing → Feature Engineering
                │
     ├─ ARIMA/SARIMA Forecasting
     ├─ VADER Sentiment Analysis
     ├─ Recommendation Models
     ▼
 Recommendation Engine
     ▼
 Flask Dashboard
```

## 📊 Evaluation Metrics
The project evaluates recommendation quality using:
- Precision@K
- Recall@K
- Mean Reciprocal Rank (MRR)
- NDCG
- RMSE
